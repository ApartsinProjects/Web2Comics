# ComicBot

Telegram bot that receives a text message or URL, generates a comic image, and sends it back.

## Easiest Setup
- Non-technical guide: `EASY_SETUP.md`
- Advanced/provisioning guide: `USER_GUIDE.md`

## Quick Start
1. Copy `.env.example` to `.env` in `comicbot/`.
2. Set `TELEGRAM_BOT_TOKEN` and provider key(s) in `.env`.
3. Start:

```bash
npm run comicbot:start
```

See `USER_GUIDE.md` for full provisioning and setup.
